- MySQL
  
## Installation
1. Clone the repository
   ```bash
   https://github.com/anngithub1234/SalonHair.git
   ```
2. Install dependencies
   ```bash
   composer install
   npm install
   npm run dev
   ```
3. Configure environment variables
   ```bash
   cp .env.example .env
   ```
4. Run Migrations
   ```bash
   php artisan migrate
   ``
5. Serve the application
   ```bash
   php artisan serve
   ```
6. Run the build
   ```bash
   npm run dev
   ```
